## Getting Started

1. Open `index.html` in a web browser
2. Start with Dashboard to see your overview
3. Go to "Today's Mission" to begin Day 1, Block 1
4. Complete blocks in order for maximum retention

## Technology

- Pure HTML/CSS/JavaScript (no frameworks)
- LocalStorage for data persistence
- Progressive Web App (PWA) ready
- Mobile-responsive design

## Training Philosophy

This app implements retention-first learning:
- Active recall over passive reading
- Spaced repetition of concepts
- Self-assessment and metacognition
- Project-based application

## Data Storage

All data is stored locally in your browser's localStorage. No data leaves your device.

To reset: Click the "Reset All Data" button in the footer (destructive action, cannot be undone).
